## 3. 候选因子定义

### trend_strength

- 含义：5 日均线相对 20 日均线的趋势强度
- 公式：`m_avg(close, 5) / m_avg(close, 20) - 1`
- 方向：higher_better
- 是否使用未来数据：False

### obv_trend_20d

- 含义：20 日 OBV 趋势强度，衡量成交量能量的持续抬升或下行
- 公式：`m_avg(m_sum(sign(close - m_lag(close, 1)) * volume, 5), 20)`
- 方向：higher_better
- 是否使用未来数据：False

### ma_gap_20d

- 含义：收盘价相对 20 日均线的位置
- 公式：`close / m_avg(close, 20) - 1`
- 方向：higher_better
- 是否使用未来数据：False

### ma_gap_60d

- 含义：收盘价相对 60 日均线的位置
- 公式：`close / m_avg(close, 60) - 1`
- 方向：higher_better
- 是否使用未来数据：False

### risk_adjusted_momentum_20d

- 含义：20 日动量除以 20 日波动率，衡量单位风险趋势收益
- 公式：`(close / m_lag(close, 20) - 1) / (m_std(close / m_lag(close, 1) - 1, 20) + 0.001)`
- 方向：higher_better
- 是否使用未来数据：False

### bbi_indicator

- 含义：多空指标 BBI = (MA3 + MA6 + MA12 + MA24) / 4，用于判断多空分界
- 公式：`(m_avg(close, 3) + m_avg(close, 6) + m_avg(close, 12) + m_avg(close, 24)) / 4`
- 方向：neutral
- 是否使用未来数据：False

### new_high_distance_60d

- 含义：当前价格距离 60 日新高的距离，越接近新高代表趋势越强
- 公式：`close / m_max(close, 60) - 1`
- 方向：higher_better
- 是否使用未来数据：False

### ma_alignment_score

- 含义：均线多头排列强度：5 日均线高于 20 日和 60 日均线时得分更高
- 公式：`(m_avg(close, 5) / m_avg(close, 20) - 1) + (m_avg(close, 20) / m_avg(close, 60) - 1)`
- 方向：higher_better
- 是否使用未来数据：False
